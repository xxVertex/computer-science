Step 1

First of all, what means HTML?

Well HTML stands for Hyper Text Markup Language.

Before starting you start coding you'll need some basic knowledge like how to create an HTML file and how to open it.

For so, you need to create a folder, with a subfolder inside, named "img", this folder will contain the images for the website.

Next, open Notepad, to open Notepad just press the Windows key and then type in "notepad".

Once you have opened Notepad, go to File>Save as... name it index.html
